# My Kernel

En la industria del software, medimos todo: latencia, tiempo de respuesta, carga de CPU y velocidad de despliegue. Sin embargo, ignoramos las métricas del motor que crea todo eso: **el desarrollador**.

El burnout y el síndrome del impostor no ocurren de la noche a la mañana; son el resultado de un Memory Leak emocional no detectado a tiempo.

**My Kernel** es un espacio seguro y minimalista diseñado para que los desarrolladores monitoreen su carga de trabajo y accedan a una base de conocimientos sobre la salud mental impulsada por profesionales y veteranos de la industria.

A diferencia de las apps de bienestar genéricas, **My Kernel** habla el lenguaje de los devs y se enfoca en el análisis de rendimiento humano sin ser intrusivo.


## Características Principales **(MVP)**
### Daily Mental Commit
Registra tu estado diario con parámetros técnicos:
- Seguimiento manual de tareas resueltas vs. esfuerzo percibido.
- Etiquetas de estado como [Stable], [Warning], [Debug Required].

### Kernel Dashboard
Visualización de datos de alto impacto. No solo gráficas, sino insights:
- Detecta si tu velocidad de resolución está cayendo sistemáticamente tras varios días de carga "Critical".
- Alertas visuales sutiles cuando los patrones indican agotamiento inminente.

### Knowledge Base (Expert-to-Dev)
Una biblioteca curada de recursos específicos para el mundo tech:
- Rutas de Bienestar como "Managing Legacy Code Anxiety", "Onboarding without Imposter Syndrome".
- Ejercicios diseñados para el flujo de trabajo de un programador.
